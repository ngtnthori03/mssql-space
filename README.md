# mssql-space
easy way to working with MSSQL

# getting started

```shell
  export NTZOE_mssql_space_DIR=$NTZOE_HOME_DIR/space/mssql;
  git clone https://github.com/ngtnthori03/mssql-space $NTZOE_mssql_space_DIR;
  sh $NTZOE_mssql_space_DIR/boot.sh
```
